Loading

基本用法

通过基础的 [message,loadIcon,appoint]设置
将message 属性赋值为 '你对应的信息'，并可通过 loadIcon 属性来指定 你想要设定的图标，还可通过appoint 属性来指定loading层位置 {注：传入你需要指定的dom元素}
<template>
  <div slot="header" style="width:100%">
    <button class="dt-button dt-button--primary" @click="toastShow(1,'')">body全局实例</button>
    <button class="dt-button dt-button--primary" @click="toastShow(2,$event)">指定dom实例</button>
  </div>
</template>
<script>
  export default {
    data () {
      return {
      }
    },
    methods: {
      toastShow (t,ev) {
        const _this = this
        if(t==1){
          _this.$loading.open({
            message: '数据加载中...',
            loadIcon: '', //完整图片路径 [static/images]
            appoint: ''
          })
        }else{
          _this.$loading.open({
            message: '数据加载中...',
            loadIcon: '', //完整图片路径 [static/images]
            appoint: ev //是否指定要使用的位置
          })
        }
        setTimeout(function(){
          _this.$loading.close();
        },1500)
      }
    }
  }
</script>
显示代码
复制代码
参数名 参数说明 参数值
message 显示消息内容 您的文本信息
loadIcon 显示图标 完整图片路径 [static/images]
appoint 显示位置 传入dom元素表示跟随当前元素显示「需要当前元素dom」
