Checker

基本用法

通过基础的属性 [bankVal,plad,maxLen,bankReplace] 设置
Checker 多选

value：[ "1" ]
Checker 最多选2个

value：[ "1" ]
Checker 单选

value：[]
将 bankReplace 属性赋值为 'true/false' 来设置是否分割，并可通过 maxLen 属性来指定输入长度，通过plad 属性 设置placeholder，bankVal来设置初始值，通过repLen属性来设置分割位数，可通过bankCall设置回调[根据使用情况而定]
<template>
  <div slot="header" style="width:100%">
     11
  </div>
</template>
<script>
    import { Checker, CheckerGroup } from "./checker/index";
    export default {
    components: {
        Checker,
        CheckerGroup
    },
    data() {
        return {
            checkboxValue: ["1"],
            radioValue: [],
            options: [
                {
                    value: "1",
                    label: "篮球"
                },
                {
                    value: "2",
                    label: "羽毛球"
                },
                {
                    value: "3",
                    label: "乒乓球",
                    disabled: true
                },
                {
                    value: "4",
                    label: "高尔夫"
                }
            ]
        }
    }
    }
</script>
