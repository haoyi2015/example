Textarea

基本用法

通过基础的属性 [maxLen,isCut,overCall] 设置
还可以输入10个字
将 isCut 属性赋值为 'true/false' 来设置是否截取，并可通过 maxLen 属性来指定输入长度（默认默认不限制），可通过overCall设置回调[根据使用情况而定]
<template>
  <div slot="header" style="width:100%">
    <limit-textarea :maxLen="10" placeholder="最多输入10个字符"></limit-textarea>
    <limit-textarea :maxLen="10" placeholder="最多输入10个字符" isCut @overCall="overCallVal"></limit-textarea>
  </div>
</template>
<script>
    import limitTextarea from './textarea'
    export default {
    data() {
        return {}
    },
    components: {
        limitTextarea
    },
    methods: {
        overCallVal (v) {
            console.log(v)
        }
    }
    }
</script>
