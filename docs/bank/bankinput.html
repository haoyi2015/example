BankInput

基本用法

通过基础的属性 [bankVal,plad,maxLen,bankReplace] 设置
将 bankReplace 属性赋值为 'true/false' 来设置是否分割，并可通过 maxLen 属性来指定输入长度，通过plad 属性 设置placeholder，bankVal来设置初始值，通过repLen属性来设置分割位数，可通过bankCall设置回调[根据使用情况而定]
<template>
  <div slot="header" style="width:100%">
    <bank-input
        :bankVal.sync="val"
        :plad="plad"
        :maxLen="16"
        :bankReplace="true"
        :repLen="4"
        @bankCall="bankCall">
    </bank-input>
  </div>
</template>
<script>
    import BankInput from './input/bank.vue'
    export default {
    components: {
        BankInput
    },
    data () {
        return {
        val: 622918911129,
        plad: ''
        }
    },
    methods: {
        bankCall (v) {
            console.log(v)
        }
    }
    }
</script>
